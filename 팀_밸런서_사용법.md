## 🧮 밸런싱 알고리즘 설명

### 1. 능력치 카테고리 분류

모든 능력치를 **5개 주요 카테고리**로 분류합니다:

| 카테고리 | 포함 능력치 |
|---------|-----------|
| **공격력** | 슛파워, 슛정확도, 골 결정력 |
| **미드필드** | 패스, 위치선정 |
| **수비력** | 수비, 몸싸움 |
| **피지컬** | 체력, 속력, 가속도 |
| **멘탈** | 정신력, 적극성 |

### 2. 조합 생성

Python의 `itertools.combinations`를 사용하여 **가능한 모든 팀 조합**을 생성합니다.

예: 8명 중 4명 선택 = C(8,4) = 70가지 조합

### 3. 밸런스 점수 계산

각 조합에 대해 다음을 계산합니다:

```python
밸런스_점수 = (블랙팀_총합 - 화이트팀_총합)의 절댓값
           + 0.5 × Σ(각 카테고리별 차이)
```

- **낮은 점수 = 더 균형잡힘**
- 총합 차이와 카테고리별 밸런스를 모두 고려

### 4. 최적 조합 선정

밸런스 점수가 가장 낮은 **상위 3개 조합**을 추천합니다.

## 📊 출력 결과 해석

### 밸런스 바

```
████████░░ 85%
```

- **█**: 균형잡힌 부분
- **░**: 불균형한 부분
- 10칸 중 8칸 = 85% 밸런스

### 밸런스 평가 기준

| 전체 밸런스 | 평가 | 설명 |
|-----------|-----|-----|
| 95% 이상 | 완벽! 🌟 | 거의 완벽한 밸런스 |
| 90-94% | 매우 좋음 👍 | 매우 균형잡힌 조합 |
| 85-89% | 좋음 ✓ | 충분히 균형잡힘 |
| 80-84% | 보통 - | 약간의 차이 존재 |
| 80% 미만 | 불균형 ⚠️ | 재조합 권장 |

### 1. 전략적 선수 고정

- **주장 분산**: 양 팀에 경험 많은 주장을 한 명씩 고정

- 1위: 가장 균형잡힌 조합
- 2-3위: 전략적 선택지 (예: 특정 선수를 같은 팀으로)
